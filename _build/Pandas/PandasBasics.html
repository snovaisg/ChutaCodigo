Pandas

This is the data manipulation library for python.

import pandas as pd

Series

Create a Pandas Series while specifying the index

pd.Series(data=["Larry", "Bill", "Mark", "Steve"], 
               index=["Google", "Microsoft", "Facebook", "Apple"])

Google       Larry
Microsoft     Bill
Facebook      Mark
Apple        Steve
dtype: object

Now using a dictionary. The keys of the dictionary will become the index of the series!

my_dict = {"Google": "Larry",
           "Microsoft": "Bill",
           "Facebook": "Mark",
           "Apple": "Steve"}

my_series = pd.Series(my_dict)
my_series

Google       Larry
Microsoft     Bill
Facebook      Mark
Apple        Steve
dtype: object

You can convert a series directly to numpy

my_series.index.to_numpy()

array(['Google', 'Microsoft', 'Facebook', 'Apple'], dtype=object)

DataFrames

Simple dataframe. Using index and column names

df = pd.DataFrame([[1,   2,   3,    7], 
                    [4.2, 6.1, 8.9, -4.1], 
                    ["a", "b", "c", "z"] ],
                    columns=["col_1", "col_2", "col_3", "col_4"],  # <-- The column names
                    index=["row_1", "row_2", "row_3"])   # <-- The row names
df
col_1 col_2 col_3 col_4
row_1 1 2 3 7
row_2 4.2 6.1 8.9 -4.1
row_3 a b c z

A more organized way to do it

company = ["Google", "Microsoft", "Facebook", "Apple"]
founder_name = ["Larry", "Bill", "Mark", "Steve"]
founder_surname = ["Page", "Gates", "Zuckerberg", "Jobs"]

df = pd.DataFrame( [ company, founder_name, founder_surname])
df
0 1 2 3
0 Google Microsoft Facebook Apple
1 Larry Bill Mark Steve
2 Page Gates Zuckerberg Jobs
